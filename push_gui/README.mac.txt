This program pushes DICOM data to the INCF server as described at
http://xnat.incf.org/.  Drag folders containing DICOM data to be
uploaded (this version will not prompt for data on its own).

Contains components from:

    PyQt4 http://www.riverbankcomputing.com/software/pyqt/intro
    DCMTK http://dicom.offis.de/dcmtk
    httplib2 http://code.google.com/p/httplib2/
    pydicom http://code.google.com/p/pydicom/
